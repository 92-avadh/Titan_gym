# Titan Fitness Club 💪

A modern premium gym and fitness website built using **Next.js 15**, **TypeScript**, **Tailwind CSS**, **ShadCN UI**, and **Framer Motion**.

Designed specifically for:

* Gyms
* Fitness centers
* Personal trainers
* Client demos
* Portfolio showcase
* Agency presentations

The project focuses on:

* Premium fitness UI
* Fast performance
* Mobile responsiveness
* Smooth animations
* Modern branding
* Conversion-focused design

# ⚙️ Installation

Clone the repository:

```bash id="e1v90q"
git clone https://github.com/yourusername/titan-fitness-club.git
```

Navigate to the project folder:

```bash id="a8u2zm"
cd titan-fitness-club
```

Install dependencies:

```bash id="v7z4l1"
npm install
```

Run development server:

```bash id="d2x6np"
npm run dev
```
